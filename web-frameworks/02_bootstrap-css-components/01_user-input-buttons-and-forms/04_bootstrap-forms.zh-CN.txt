接下来让我们学习一下 Bootstrap 对表格的支持 在 bootstrap 中 来实现不同风格的表格 我们之前已经学习过了许多和表格有关的知识 所以在这节课 我会着重讲解 Bootstrap 的各种类以及对应的不同风格的表格 在 Bootstrap 中 表格是通过 form 标签来创建的 表格中的所有元素都会被封装在
一个 form 标签下 如果我们不对表格指定任何一个类 那么 Bootstrap 会选择应用默认格式的表格 我们可以用 form-horizontal 来定制一个表格 如果我们这样做了 那么标签将会 在输入框的左侧显示 正如这个例子所示 我们用 form-group 同时实现了一个标签和一个输入框 并将它们封装在了一起 这里我将标签内容设置为了 First Name 它显示在了表格的左侧 而右侧的同一水平线上 则是一个输入框 于是我们发现 Bootstrap 默认设置中 form-group 是拥有对 row 类的支持的 所以在 form-group 里面我们可以使用 column 类 来排列好 form-group 中的元素 你可以看到我对标签和输入框 都应用了 column 类 我们还可以定义同行表格 同行表格将会在一行内显示其所有内容 这是很有用的功能 比如说 它可以让你的导航栏 支持用户在其中输入账户密码来登录 行内表格是通过 class="form-inline" 来实现的 在其中你可以指定一些 form-group 而在这个栗子里 我对一些 label 标签 指定了一个 sr-only 类 表示这些标签只有在用户使用屏幕阅读器时才能看到 对于一般的情况 这些标签将会被隐藏 而作为代替 我们可以使用 placeholder 属性 来告诉用户这个输入框是干嘛用的 另一个 Bootstrap 支持的类叫 input-group 它允许你将一系列元素组合起来 而且对于你表格中的任何一个输入框 你都可以用 input-group-addon 来在其左或右侧 加入一些元素 在这个例子里 我就在用户输入地区编码的输入框左右 加了一对括号 这个地区编码可以用来加在电话号码前面 你可以看到 我同时对左括号和右括号 应用了 input-group-addon 需要注意的是 这些类需应用到一个 div 中 接下来我们需要完成一个练习 在这个练习里我们将会在我们的网页中 加入一些按钮 和表格来允许用户 给我们的餐厅做出反馈
