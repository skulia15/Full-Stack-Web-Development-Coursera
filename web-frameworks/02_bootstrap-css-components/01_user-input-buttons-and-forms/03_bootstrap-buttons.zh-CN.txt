现在让我们说说 Bootstrap 的按钮 事实上 Bootstrap 提供了很多和按钮有关的类 从而你可以在你的网页中实现许多不同的按钮 还可以在许多表格中运用不同风格的按钮 Bootstrap 按钮的默认设置中有
五种不同的颜色 这些颜色可以通过 设置不同的类来实现 比如 primary 代表深蓝色 info 代表浅蓝色等等 Bootstrap 里默认的类格式大部分为
xxx-yyy 其中 xxx 是你的对象类别
比如说 text 或者 btn 或者 bg yyy 则是 primary, success, info, warning, danger
中的一个 各自对应的颜色如图所示 这五种颜色 基本可以用在所有场合 举一些栗子 如果你想要天蓝色的 text 那么就设置 text-info 如果你想要一个绿色的按钮 就设置 btn-success 如果你想要一个红色的警告 就设置 alert-danger 类似的 如果你想要一个天蓝色的背景 就在相应元素中设置 bg-info 这五种颜色的组合 可以用在几乎所有 你需要指定颜色的地方 我们之前说过 Bootstrap 提供了许多按钮的类 它们允许你在不同场合使用不同风格的按钮 现在&nbsp;你可以将这些按键类应用到<a>标签
或<button>标签中 又或应用到你们网页中的<input>标签中 它们都可以通过使用不同的 button 类
来获得不同风格的按钮 当然 当你对这些标签应用 Bootstrap 的按钮类时 它们都会长得像一个按钮 不过需要注意的是 只有 button 标签才能被应用到 navs 和 navbars 中 让我们来看一些栗子吧 如图所示 我们在左边定义了五种不同的按钮 那个很大的按钮是用了 btn, btn-primary btn-lg 来设置的 而从左到右的按钮 对应到代码中 则是从上到下 逐一对应 而那两个定义在右边的按钮 用了一个叫 btn-block 的类 这将会使这些按钮舒展开 并在宽度上占据整个相应板块 在这个例子里 这两个按钮都占满了所有的列 也就是说占满了三列 综上所述 这些 Bootstrap 中的按钮 其实都是这四种不同设置的组合 当你想要实现一个按钮时 先指定其为 btn 然后选择颜色 这里有 7 种选择 继而是选择大小 你可以选择大 中或小 当然你也可以不指定大小 那么 Bootstrap 将会帮你选择默认的按钮大小 最后就是看要不要设置 btn-block 也就是看你想不想让按钮横向充满
当前的整个板块 Bootstrap 还允许你将按钮塞进
同一个按钮组中 当你确实这么做了的时候 这些按钮就会以一个单元的形式显示 如图所示 我们实现了三种不同样式的按钮组 左边两个都被 封装在了一个 btn-toolbar 中 那么怎么实现按钮组呢 直接对一个 div 设置 class="btn-group" 即可 同时如果你需要的话 你可以通过
btn-group-sm 等来设置按钮组的大小 你还可以设置按钮组的排列方向 通常来说是按钮组中的按钮都是 横向排列的 但你也可以设置为纵向排列
也就是 btn-group-vertical 这样那些按钮就会叠在一起 正如图中右边的按钮组所示 我们将 info 按钮 warning 按钮和 danger 按钮叠在了一起 而代码的实现也可以在幻灯片上看到 最后让我们来总结一下 首先是 btn 类 然后是颜色 然后是大小 按钮可以用 btn-toolbar, btn-group, btn-group-vertical btn-group-justified 等来组合在一起 btn-group-justified 代表按钮将会 占据所有可能占据的空间 所以如果它们被包含在了一个列里面
它们将会充满整个列 当然你还可以通过 lg, sm, xs 来设置 btn-group 的大小
