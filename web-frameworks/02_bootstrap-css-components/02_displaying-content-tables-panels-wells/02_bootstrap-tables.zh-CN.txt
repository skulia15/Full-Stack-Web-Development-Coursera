接下来让我们把普通的表格转化成
Bootstrap 的表格 我们将学习如何用 Bootstrap 类来定义各种表格 当表格最初被引入到 HTML 中时 它们使用来在网页上呈现表格状的数据的 然而有时表格会在许多其它场合下被"盗用" 在一些完全和呈现一张表格数据无关的场合 我们已经见过人们 用表格来给它们网页上的内容排版 不过当时的我们确实没有其它的办法 来控制这些内容的排版 现在 由于有了像 Bootstrap grid
这样的网格系统 我觉得我们就不应该再用表格 来进行内容的排版了 事实上我很讨厌这种做法 不管怎么说 我们还是讨论一下
表格最初的用途吧 亦即 在网页上呈现表格数据 HTML 里的 table 标签当然是用来显示
一个表格数据的 而 Bootstrap 所提供的一个叫 table 的类 则是用来应用到 HTML 的 table 标签上的 你可能会想 为什么我们除了 table 标签外
还需要 table 类呢 事实上 一个 table 标签仅仅代表了一个表格而已 而 Bootstrap 的 table 类则会让这个表格
符合某种样式 这个样式在显示表格数据时 毫无疑问
是相当漂亮的 也就是说如果你想快捷方便地创建一个漂亮的表格
用 table 类将是一个很好的选择 因此我们常常看到 Bootstrap 中的表格 用 table 类来设计版面 当然除了 table 类 在 Bootstrap 中还有很多其它的类
可以拿来定制表格 比如说 table-striped table-borderd, table-hover, table-condensed,
table-responsive 等等 而在我们的练习里 我们将会
学习 table-striped 的使用 这个类会创建一个背景颜色灰白相间的表格 意味着这一行的背景颜色是浅色的话
上一行和下一行的背景颜色则会是深色的 这样就能更简单地区分表格中的各个行 table-bordered 类则会给表格添加边框 所以如果你需要一个带边框的表格的话 你就可以使用 table-bordered 类 table-hover 类 顾名思义
会对你的鼠标位置作出响应 当你用鼠标移动到表格中的某一行时 那一行就会被高亮显示 而如果你想减少单元格边距的话 你可以使用
table-condensed 类 而这也是 Bootstrap 表格中的标准类 你还可以用 table-responsive 来创建一个
响应式表格 也就是这么一个表格 即使它处于
不同的页面大小中 它也能很漂亮地进行其内容的呈现 比如在一个小于 768px 的很小的屏幕中时 响应式表格将会自动具有横向滚动的功能 因此 如果你的响应式表格不能适配当前屏幕时 Bootstrap 的 table-responsive 将会支持滚动功能
从而用户可以更好地阅读其内容 除此之外 我们还可以通过五个类 来给表格的行或者单元格着色 这些类就是 active, success, info, warning
和 danger 我们之前其实已经学习过这五个类了 所以我们可以类似地将这些类应用到表格的行 或是单独的单元格中 以上 就完成了对表格和 Bootstrap table 类的讨论
