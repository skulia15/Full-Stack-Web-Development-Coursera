Web Tools: Bower: Objectives and Outcomes

In this lesson we will learn more about the command-line web tools that make use of the Node runtime to provide a lot of valuable support for web developers. These tools are written in JavaScript and enable web developers to automate a number of repetitive tasks that need to be done during web development. At the end of this lesson you will be able to:

  • Understand the need for web tools

  • Make use of Bower to automatically fetch Web packages and assets.
