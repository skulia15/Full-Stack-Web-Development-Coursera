[]让我们现在来学习一下如何在我们的电脑里面 部署 Node.js 和 NPM 首先去往 nodejs.org 网页 然后单击这个 Download 按钮 来下载 Node.js 的安装器 当你完成下载后 双击这个安装器并开始安装的步骤 现在我用我的 Mac 来演示一下 安装的过程 如果你用的是 Windows 或者 Linux 的话 你的安装窗口可能会稍有不同 不过都是类似的 你只需要按照安装器给你的提示和建议
一步步地进行安装即可 当你完成了安装之后 为了检查 Node 是不是
确实被安装到了你的机器上 若你用的是 Mac 或 Linux 你可以打开 terminal 如果你用的是 Windows 的话 就打开命令行 然后在里面输入 node -V 来检查 你机器中所安装的 node 的版本 类似的 你可以通过输入 npm -v 来检查 你机器中安装的 npm 的版本 如果这两个命令都能正确执行的话 你的机器就已经能用 Node.js 了
