理解响应式设计的确切含义 以及你该如何把网站设计为响应式 那么马上问题就来了 为什么我们要考虑响应式设计呢 ? 为什么我们要网站设计成响应式的呢 ? 当我们观察时下的网站是如何被访问的时候 你会发现人们从各种不同的设备访问网站 从台式机 笔记本电脑 平板电脑到手机都有 所以 你的网站有可能被用户
以各种不同尺寸的屏幕浏览 如果你设计网站时只考虑一种尺寸 那么它就有可能被不正确地渲染 又或者在屏幕小的设备上完全不可读 因此如果我们可以主动地设计网站 使其能灵活地适应浏览者的屏幕大小 浏览者就能更轻松地 清楚地浏览你的网站 现在 如果我们可以将这个特性构建在网站本身 让它能够自动调整布局去适应用户的屏幕尺寸 那将会是一大改进 响应式设计的目标就是在实践中实现这一目标 要做到这一点 我们要用到设备的视域 (viewport) 属性 来调整我们的网站到设备的屏幕上 理解响应式设计的最好方法 是看一看例子 我们看看这个课程网站本身 你可以看到浏览器窗口开着课程网站 我可以将它展开到整个屏幕的宽度 这个宽度现在是1280像素 设想我要用一个设备来浏览同一个网站 这个设备的屏幕只有大概700像素 这种情况下网站应该如何被渲染呢 为了帮助更好地我们理解 我要做的是缩小浏览器的尺寸 让网站自动调整到浏览器的尺寸 这样 从某些方面来说 我就是在模仿 从不同尺寸的屏幕访问网站的效果 现在 屏幕的宽度是1200像素 这相当于时下的一台笔记本电脑的典型尺寸 让我们看看用平板电脑的横屏状态 来访问这个网站时的效果 也许这样看起来比较像平板电脑的屏幕尺寸 典型平板电脑的横屏尺寸大约在700像素左右 有些设备的尺寸会更大 而较旧的七吋笔记本电脑 可能会有800像素左右的屏幕宽度 所以 这就是你的网站在平板电脑上看起来的样子 让我们继续缩小浏览器窗口 看看网站在更小的尺寸上看起来是怎样的 当你继续缩小窗口尺寸的时候 留意网站的内容 会自动适应屏幕的尺寸 现在 留意当我将窗口缩小到某一个特定大小的时候 网站在浏览器窗口中显示的方式会发生一个突然的变化 我们来看一下 我们在这个尺寸 然后只要再缩小一点点 你会突然发现网站渲染得不一样了 你可以看到窗口顶部的导航栏已经消失 被左侧的一个按钮代替了 现在导航栏依然是可访问的 当你点击那个按钮的时候 会弹出一个侧边栏 上面的信息 就是刚才较大的屏幕上 网站侧边栏上的信息 现在侧边栏在这个屏幕尺寸上被隐藏了 我可以点击任意其他位置来隐藏这个侧边栏 另外 留意网站的内容 也自动调整到屏幕的尺寸 这就如同平板电脑 在横向摆放时的效果 让我们继续缩小网站窗口的尺寸 你可以看到网站内容是如何自动调整的 当我们滚动网站的时候 你可以看到它的内容已经调整到 浏览器窗口的尺寸 或者说 相当于屏幕的尺寸 这是怎么发生的呢 这就是响应式设计的威力了 另外一个你常常会听人提起的 响应式设计范畴内的术语是移动优先 确切来说 什么是移动优先式设计? 传统上 当人们设计网站 需要这个网站同时兼顾 大屏幕设备 如笔记本电脑 台式电脑
和小屏幕设备时 他们常常会先设计大屏幕下的布局 然后再尝试调整网站的内容 使其在小屏幕上也能恰当地显示 因此在这幅图中 传统的方法会是 先给笔记本电脑设计 再调整内容 去适应更小的窗口尺寸 现在 人们越来越多地采用移动优先的方法 意思是你设计网站时 首先考虑的是 移动设备的屏幕尺寸 首先为移动设备的屏幕设计网站布局 之后再为更大的屏幕尺寸调整网站设计 这样 当你的屏幕尺寸扩大时 内容会自动流动 并调整到占据更大屏幕的设备 所带来的更宽的屏幕宽度 为了达到移动优先式设计 需要有三样东西的支持 第一 你的前端界面框架应该支持网格系统 它已经越来越多地被大部分前端界面框架所支持 因此当我们观察任意一个通行的前端框架 都会发现它以某种形式支持网格系统 我们会在下一堂课里更详细地 探讨网格系统 尤其是Bootstrap网格 (Bootstrap Grid) 第二方面是要有流动图像 又称为响应式图像的支持 这样你就可以 使网站上的图片自动根据屏幕尺寸调整大小 这又是一个必须内嵌的特性 Bootstrap 已经在它的 CSS 类中为流动图像 或称响应式图像提供了很好的支持 我们会在下一章中更详细地探讨这个话题 第三种我们所需要的支持叫做媒体查询 (media queries) 如果你还记得 在前一张幻灯片里 我提到了视域 通过视域 你可以大致感受到 用来浏览你网站的屏幕的尺寸 对于移动设备来说 如果恰当地使用视域 你应该能够发觉屏幕屏幕尺寸相当小 只有约700像素 甚至比这更小 而对于更大的屏幕尺寸来说 很明显 你可以调适 通过在CSS类的设计中加入媒体查询并且加以利用 视域会很有帮助 在实践中这要如何达成呢 媒体查询通过这样的方法实现 在你的CSS文件中 根据不同的窗口尺寸 你可以将每个CSS类定义得稍有不同 要进行一次媒体查询 你要在你的CSS文件中加入像这样的一条语句 你会写 @media 和 screen 然后指明最小的宽度 然后在这条媒体查询的花括号内 你可以列举一些CSS样式 这些样式 只会在屏幕宽度在600像素以上时才适用 像这样你就可以设计不同的CSS代码给不同的屏幕尺寸 例如 在Bootstrap里 就像我们稍后会看到的那样 有的媒体查询会重新设计CSS类 有的则会将CSS类因应不同的屏幕宽度作调整 为了帮助我们更好地理解 我打开了bootstrap.css文件 这个文件曾经用在我们之前设计的项目中 我用Brackets打开bootstrap.css文件
