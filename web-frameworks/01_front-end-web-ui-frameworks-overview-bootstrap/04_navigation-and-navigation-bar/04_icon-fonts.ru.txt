Позвольте провести краткий обзор
графических шрифтов, что это такое,
как они используются на веб-сайтах и, какие преимущества они дадут, если
вы будете использовать их на своём веб-сайте. Графические шрифты - это набор иконок и символьных знаков. Они могу быть использованы также,
как и обычные шрифты на вашем сайте и, подобно тому, как вы используете стандартные
текстовые шрифты на вашем сайте, вы можете использовать графические шрифты в
любом месте сайта. Преимущество их использования заключается в том,
что к ним могут быть легко применены стили CSS. Таким образом вы можете управлять цветом,
размером, фоном и многими другими параметрами
графических шрифтов, используя CSS. Графические шрифты гораздо легче по
сравнению с графикой на ваших веб-страницах. Так что, если у вас есть простая графика,
то вместо использования изображений, вы можете работать с графическими шрифтами,
как с простой заменой изображениям. В случае, когда ваш веб-сайт
содержит графические шрифты, они загружаются без необходимости
дополнительных запросов, которые нужны в случае
загрузки изображений. И это одна из причин, почему графические шрифты завоевали огромную пополярность при использовании их на веб-сайтах. Множество различных графических шрифтов доступно,
так как они используются дизайнерами веб-сайтов. Чтобы понять, как графические шрифты
используются на веб-сайтах, давайте коротко взглянем на
веб-страницу вашего курса. Итак слева на панели вы можете увидеть, что у нас
есть несколько ссылок Кроме самих ссылок на панели
в левой части страницы мы видим также графические иконки,
включённые в веб-страницу. Все эти иконки были созданы
с использованием графических шрифтов. Обратите внимание на несколько примеров
используемых графических шрифтов. В частности, в рамках этого курса, мы будем
использовать библиотеку шрифтов Glyphicons. Библиотека Glyphicons уже включена,
как часть фреймворка Bootstrap. Вдобавок, Также мы будем использовать ещё одну популярную
библиотеку графических шрифтов Font Awesome. Как я уже говорил, библиотека Glyphicons
уже включена в Bootstrap. Для использования Glyphicons в любом месте вашей
веб-страницы нужно воспользоваться тегом span. Для тега span задайте класс с именем glyphicon и затем специфическое имя класса для иконки,
которую вы собираетесь использовать в этом месте. Как вы видите, здесь я использую glyphicon-home
в качестве второго имени класса. Имя класса glyphicon-home связано с
иконкой дом в наборе Glyphicon. Библиотека Glyphicons, включённая в Bootstrap,
включает в себя около 250 иконок. Другой популярный графический шрифт,
который я упоминал, и который мы будем использовать в рамках
данного курса - это Font Awesome. Сейчас Font Awesome включает в себя
большой набор иконок. И предоставляет широкий выбор иконок,
которые вы можете использовать на своём сайте. Теперь, когда вы увидите иконки из Font Awesome,
вы непременно скажете: "Вот это потрясающе!" Хорошо, хорошо. Вы можете смеяться сколь угодно. Я совершенно не собираюсь выражаться,
как подросток. Хорошо, давайте вернёмся к
графическим шрифтам. Чтобы использовать эти иконки на веб-сайте
воспользуйтесь тегом "i". Для тега "i" задайте имя класса fa,
затем имя класса fa- и специфическая иконка, которую
вы планируете использовать в данном месте. И так в этом примере вы можете видеть,
что я использую fa fa-phone - иконку телефона, которая будет установлена в
определённое место на вашей веб-странице. Как вам начать использовать библиотеку
Font Awesome в вашем веб-проекте? Вам нужно скачать дистрибутив
Font Awesome, и затем вам следует включить файлы CSS, которые
входят в Font Awesome, в ваш веб-проект. И все шрифты будут
автоматически подключены. Есть одно действие, которое я выполняю
при использовании Font Awesome в моих веб-проектах - после
скачивания дистрибутива Font Awesome я копирую файлы CSS из папки CSS
дистрибутива Font Awesome в папку CSS в моём веб-проекте. Также я копирую шрифты из
дистрибутива Font Awesome в папку fonts в моём веб-проекте. Я использую это таким образом. Вы можете включить файлы из Font Awesome
так, как вы захотите. Другой CSS-класс, который я собираюсь
использовать в задании, которое последует за лекцией -
это Bootstrap-social. Bootstrap-social - это CSS-класс,
который позволяет оформить некоторые кнопки
для социальных веб-сайтов. Bootstrap-social использует графические
шрифты Font Awesome's так, чтобы отображать логотипы для
различных социальных веб-сайтов. Недавно Bootstrap-social добавил новые
возможности для шрифтов Font Awesome. Например он берёт иконки
из Font Awesome и превращает их в кнопки для
различных социальных веб-сайтов. Как вам начать использовать Bootstrap-social? Скачайте Bootstrap-social CSS-файл и
затем включите его в вашу веб-страницу. После этого, там где необходимо
установить кнопку для социальных веб-сайтов, просто используйте тег "a" и
имя класса btn. btn, это разумеется, кнопки, которые мы
подробно рассмотрим в следующем модуле. Затем вы добавляете имя класса
btn-social-icon и имя класса для специфического
социального веб-сайта. В этом примере я использую кнопку 
для Facebook, затем вам нужно будет добавить
иконку из Font Awesome для нужного вам социального веб-сайта. Здесь я использую тег "i" и имена классов
fa и fa-facebook для отображения иконки Facebook,
которая включена в набор Font Awesome. Мы можем двигаться дальше к
